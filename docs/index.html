Getting started

Spin up Kong and other service container using docker compose.#

git clone https://github.com/pravin-raha/kong-playground.git
cd kong-playground
cd docker
docker-compose up

Stop docker compose#

docker-compose stop

Tear down the complete Kong infrastructure.#

docker-compose down -v
